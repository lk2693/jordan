# Jordan GmbH - Bilderverwaltung

Dieser Ordner enthält Bilder für die Jordan GmbH Website.

## Aktuelle Bildverwendung

Die Website nutzt derzeit hochwertige Unsplash-Bilder als Platzhalter:

### Hero-Section
- Hintergrundbild: Heizungs-/Sanitärinstallation
- Hauptbild: Professionelle Installationsarbeiten

### Services
- Wärmepumpen: Moderne Wärmepumpentechnologie
- Heizungsanlagen: Heizkessel und Heizsysteme
- Sanitär & Bad: Moderne Badinstallationen
- Klimaanlagen: Klimatisierungssysteme
- Elektroinstallation: Elektrische Installationen
- Notdienst: Notfall-Service-Einsatz

### Features
- Fachkräfte: Erfahrene Handwerker
- Material: Hochwertige Ausrüstung
- Zufriedenheit: Glückliche Kunden

### Expert-Sektion
- Installation: Fachmann bei der Arbeit

## Für Production empfohlen

Ersetzen Sie die Unsplash-Platzhalter durch eigene, authentische Fotos:

1. **Eigene Projekte**: Fotos von tatsächlichen Installationen
2. **Team**: Portraits Ihrer Mitarbeiter
3. **Firmenfahrzeuge**: Branded Servicefahrzeuge
4. **Zertifikate**: Offizielle Zertifizierungen
5. **Referenzobjekte**: Abgeschlossene Projekte

## Bildformate & Optimierung

- **Logo**: SVG (skalierbar)
- **Fotos**: WebP oder JPEG (optimiert für Web)
- **Icons**: SVG
- **Empfohlene Größen**:
  - Hero: 1920x1080px
  - Service-Karten: 800x600px
  - Features: 400x400px

## Next.js Image Component

Die Website nutzt Next.js Image-Komponente für automatische Optimierung:
- Lazy Loading
- Responsive Bildgrößen
- Moderne Formate (WebP)
- Performance-optimiert
